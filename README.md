# UI/UX Design Task 2 – Home Services App

This repository contains the submission for **UI/UX Design Task 2**, which focuses on designing a modern **Home Services Mobile App interface** that allows users to easily discover and book household services.

## Project Overview
The project presents a **Home Services App UI/UX design** created to simplify the process of booking everyday services such as cleaning, plumbing, electrical repair, and maintenance.  
The design focuses on a **clean layout, intuitive navigation, and a seamless user experience**.
